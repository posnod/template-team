# Project Template Posnod

Repo ini berisi template dasar untuk tim:

- **Frontend:** Next.js + Tailwind + shadcn
  (`frontend/`)
- **Backend:** Laravel + PostgreSQL
  (`backend/`)
- **Panduan Tim:**
  Lihat `docs/team-guidelines.md` untuk workflow, naming, testing, linting
- **Testing:**
  `test/frontend/` dan `test/backend/`

- **Dev Environment:**
  Gunakan `docker-compose.yml` jika ingin setup dev environment cepat
